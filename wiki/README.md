## Product Vision
Concordia seeks to redefine the experience of co-living.  Rather than just a task manager, it aims to become a way to live together in fairness, awareness, and ease.  Because when home feels balanced, everything else falls into place.

## Epics
### Scrum
#### As a household member, I want to create and manage tasks within a Scrum board, so that I can stay organized and coordinate responsibilities effectively.
- As a group member, I want to create a task with a title, description, and deadline so that everyone knows what needs to be done.
- As a group member, I want to assign a task to a specific user so that responsibility is clear.
- As a group member, I want to edit or delete tasks so that I can keep the board up to date.
- As a group member, I want to view tasks organized by status (“To Do”, “In Progress”, “Done”) so that I can track our progress.
- As a group member, I want to view a list of completed tasks so that I can review what has been done and by whom.
- As a group member, I want to view and categorize tasks by estimated size, so that I can understand the expected effort or duration of each task.

#### As a household member, I want to receive notifications when a task’s status changes, so that I stay informed and up to date.
- As a group member, I want to receive a notification when a task is completed so that I can stay updated on group progress.
- As a group member, I want to receive a notification when a task is assigned to me so that I know when I have new work to do.

### Money Management
#### As a household member, I want to access a overview of expenses, so that I can track spending.
- As a group member, I want to record an expense with amount, payer, description, and category so that we can track household spending.
- As a group member, I want to view a list of all expenses so that I can easily monitor the group’s spending.
- As a group member, I want to see a summary of total and individual expenses so that I can understand our overall financial situation.

#### As a household member, I want to manage a shared overview of expenses, so that I can ensure fair cost distribution.
-  As a group member, I want to adjust each member’s percentage share of an expense so that uneven contributions can be reflected accurately.
-  As a group member, I want to see how much each person owes or is owed so that we can settle balances easily.
-  As a group member, I want expenses to be automatically split among all members so that we don’t have to calculate shares manually.

### Stock Management
#### As a household member, I want to manage the house’s pantry through the app, so that I can keep track of inventory and ensure we have necessary supplies.
- As a group member, I want to add products to the pantry with name, quantity, and optional expiration date so that we can track available items.
- As a group member, I want to view a full list of pantry products so that I know what we currently have.
- As a group member, I want to quickly increase or decrease a product’s quantity in the pantry, so that stock levels stay accurate.
- As a group member, I want to be notified when a product is low in stock or nearing expiration so that we can avoid running out or wasting food. (MVP?)

#### As a group member, I want the app to automatically generate a shopping list based on missing items so that we can restock easily.

### Account and groups
#### As a user, I want to create an account in the app and manage my information
- As a user, I want to register an account with credentials (email + password) so that I can create a unique identity in the system.
- As a user, I want to log in using my credentials so that I can securely access my personal and group data.
- As a user, I want to reset my password through a secure token-based flow so that I can regain access if I forget my credentials.

#### As a user, I want to manage and join groups in the app
- As a user, I want to create a household group that acts as a shared workspace for tasks, expenses, and stock management.
- As a user, I want to invite other registered users to join my group so that they can collaborate with me.
- As a user, I want to be invited to join groups, so that I can collaborate with other users
- As a group member, I want to leave a group so that I no longer have access to its resources.
- As a group member, I want the system to restrict access to my group’s data so that only authorized members can view or modify it.
