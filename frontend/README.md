## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Run the development server:
   ```bash
   pnpm run dev
   ```

3. Open [http://localhost:8080](http://localhost:8080) in your browser.

## Build for Production

```bash
pnpm run build
pnpm run start
```

## Linting

```bash
pnpm run lint
```
